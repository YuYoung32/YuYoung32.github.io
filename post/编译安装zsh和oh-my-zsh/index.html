安装ZSH

可以通过包管理器直接安装，yum install zsh或apt-get install zsh。

centos上版本过低，需要编译安装。

预先安装编译需要的库。

yum install git make ncurses-devel gcc autoconf man

得到需要的zsh，git clone或直接手动上传。

git clone --depth=1 https://github.com/zsh-users/zsh.git zsh-folder
cd zsh-folder

配置与编译

./Util/preconfig
./configure
make
make check
make install

验证安装

zsh --version

使用zsh

exec zsh # exec替换当前进程

参考：zsh从源码编译安装 - 我的学习生活总结 (tjuyy.cn)

安装Oh-my-zsh

Oh-my-zsh和zsh的关系

zsh是一种shell，而Oh-my-zsh是一个社区驱动的框架，用于管理和扩展zsh的功能和配置。

官网Oh My Zsh - a delightful & open source framework for Zsh

方式1

sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

方式2

sh -c "$(wget https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh -O -)"

修改~/.zshrc里的ZSH_THEME="agnoster"

source ~/.zshrc

生效
